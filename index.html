Page web pour allumer une lumière

Tâches réalisables par cette page:

  • Allumer/Eteindre la lumière
  • Ajouter un thème clair ou sombre en fonction du choix de l'utilisateur

Comment fonctionne cette page :

  • Cliquer sur le bouton à gauche du titre de la page pour changer le thème principal de la page
  • Cliquer sur le bouton "Allumer" pour allumer la lumière
  • Cliquer sur le bouton "Eteindre" pour éteindre la lumière
